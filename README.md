### AI StudyMate Assistant

# Overview

AI StudyMate Assistant is an AI-powered educational document intelligence system developed using Retrieval-Augmented Generation (RAG) architecture. The application enables users to upload one or more PDF documents, perform semantic search, and receive context-aware answers generated by a Large Language Model (LLM).

The project combines document processing, vector embeddings, semantic retrieval, and conversational AI to create an intelligent study assistant capable of understanding educational content and answering questions based on uploaded documents.

---

# Project Objective

Students often spend significant time reading lengthy study materials, searching for important information, and understanding complex concepts.

AI StudyMate Assistant addresses these challenges by:

- Processing educational PDF documents
- Understanding document content using AI
- Performing semantic search across documents
- Providing contextual answers to user questions
- Supporting conversational interactions for better learning

# Workflow

Step 1

Upload one or more PDF documents.

Step 2

Extract text from uploaded documents using PyPDF.

Step 3

Split extracted text into smaller chunks using RecursiveCharacterTextSplitter.

Step 4

Generate embeddings for each chunk using Sentence Transformers.

Step 5

Store embeddings in a FAISS vector database.

Step 6

Accept user questions through the chat interface.

Step 7

Convert the question into an embedding vector.

Step 8

Perform semantic similarity search to retrieve relevant chunks.

Step 9

Send retrieved context and user query to Groq LLM.

Step 10

Generate and display a contextual AI response.
